# Global Deforestation and Forest Cover Trend Analysis Using Python

## Project Overview
This project analyzes global forest area trends using World Bank forest area and GDP growth datasets. The analysis focuses on identifying long-term forest cover changes, countries with major forest loss/gain, and the relationship between GDP growth and forest area.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Wide-to-long data transformation
- Global forest area trend analysis
- Top countries by forest area
- Largest forest loss analysis
- Largest forest gain analysis
- GDP growth vs forest area correlation analysis

## Key Findings
- Global forest area declined steadily from 1992 to 2021.
- Russian Federation and Brazil have the largest forest areas.
- Brazil experienced the highest forest loss since 1990.
- GDP growth showed a very weak negative correlation with forest area.
- Forest area change is influenced by multiple factors beyond GDP growth.

## Conclusion
The project highlights long-term global forest decline and shows that deforestation is a complex issue influenced by policy, economic activity, land use, and conservation efforts.
